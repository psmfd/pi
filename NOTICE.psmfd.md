# PSMFD mirror notice scaffold

The upstream pi project is licensed under the MIT License.

This file is a PSMFD mirror overlay scaffold for redistribution notices associated with PSMFD-built release artifacts. It will be populated or superseded by the release SBOM/NOTICE process once the build-and-attest pipeline is implemented.

Until that pipeline is implemented, this file does not replace upstream license files or third-party notices present in the upstream source tree.
